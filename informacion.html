Información de Desarrollado de la web Encriptador


                Soy    Andres   Muñoz   Taborda   de  Colombia  y  esta  mi  primera  pagina  web.
                Este es    un proyecto web   desarrollado  para el challenge   encriptador      de
                Alura  Latam    y Oracle   One Next Education, en cual se  usaron tres tecnologías
                de desarrollado web: html, css  y javaScript.

                En   html   que   es estructura    se   busco   un   enfoque    semántico  para la 
                legibilidad  de  esta  misma,  se usaron etiquetas como header  para el encabezado,
                nav    para    la  barra  de  navegación,  section para dividir  secciones y footer
                para  el  pie  de  pagina  y otros tipos de etiquetas como de link; 

                En  el   css  se  usaron  diferentes  tipos  de selectores de id, class y etiqueta
                para   llamar  los  elementos  del  html,  también    propiedades    diversas como 
                de  dimensión   height,  width;   de   posición  top, right y propiedades de color 
                y demás.

                En  javaScript  se  reutilizo  código del curso lógica de la programación de Alura
                Latam   (function asignarTextoElemento()),  se  crearon dos funciones de encriptar 
                y   desecriptar  usando el método replace() para remplazar los caracteres o string 
                del texto ingresado para  encriptar o desencriptar,    también para cumplir con el
                objetivo   de   la   regla de solo minúsculas, a las    funciones   encriptado   y 
                desencriptdo  se   creo  una funcionalidad de mas para que puedan determinar si el
                texto ingresado a procesar es solo en minúsculas, para esto se creo   un array con 
                todo  el    abecedario   en   minúsculas y   también   espacios  de 1 a 3, se creo 
                un  ciclo mediante for para iterar el string ingresado, se uso la variable inicial 
                del  ciclo for como  indice para obtener cada carácter usando el método charAt() y
                luego con eso se uso el método include() a nuestro  array de abecedario minúsculas 
                y  espacios  (caracteres permitidos)  para  determinar  si  el carácter iterado en 
                del string  ingresado  por  el  usuario  esta  en  el array, asi   nos devuelva un 
                valor  boolean  y  usar ese  valor como punto de comparación para una condición if 
                para  comprobar  si  cada carácter es permitido y asi devolver un texto encriptado
                al usuario o  una advertencia de solo caracteres permitidos.
